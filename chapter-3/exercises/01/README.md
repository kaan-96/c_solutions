## Exercise 1
What output do the following calls of `printf` produce?</br>
(a) `printf("%6d,%4d", 86, 1040);`</br>
(b) `printf("%12.5e", 30.253);`</br>
(c) `printf("%.4f", 83.162);`</br>
(d) `printf("%-6.2g", .0000009979);`

### Solution
(a) `••••86,1040`</br>
(b) `•3.02530e+01`</br>
(c) `83.1620`</br>
(d) `1e-06•`</br>
Note: "•" denotes a space in the output.
