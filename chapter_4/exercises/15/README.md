## Exercise 15
Give the values of `i` and `j` after each of the following expression statements has been executed. (Assume that `i` has the value 1 initially and `j` has the value 2.)

(a) `i += j;`</br>
(b) `i--;`</br>
(c) `i * j / i;`</br>
(d) `i % ++j;`

### Solution
(a) 3</br>
(b) 3</br>
(c) 2</br>
(d) 2</br>
